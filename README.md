## Project Description

Stackloom is a developer intelligence platform that helps engineering leaders, library maintainers, and technical researchers evaluate the real-world health, adoption, and support surface of open-source packages before they commit to them.

It cross-references GitHub repository activity and social signals with npm registry metadata and StackExchange question volume to build a multi-dimensional risk and momentum profile for any package or ecosystem. The result is a searchable intelligence layer for scorecards, trend charts, comparison views, dependency impact analysis, and alertable signals.

## Features

- Package health scorecards and ecosystem intelligence
- GitHub, npm, and StackExchange signal aggregation
- Comparison views for evaluating multiple packages
- Trend charts for downloads, releases, and support activity
- Alert rules for maintenance lag, bus factor risk, and support debt
- Search and filtering across packages and ecosystems
